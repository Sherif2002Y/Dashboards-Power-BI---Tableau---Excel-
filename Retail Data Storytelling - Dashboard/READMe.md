## Project Overview

This applied project represents the transition from learning fundamentals to delivering a fully integrated, decision-driven Tableau solution.

After completing an intensive **Data Analysis Diploma at Epsilon AI** and mastering **Power BI fundamentals**, this project demonstrates adaptability by applying advanced analytical and visualization skills using **Tableau**.

The project focuses on **Retail Analysis**, structured into two main dashboards:
- Sales Dashboard  
- Customer Dashboard  

The objective is to transform raw retail data into an interactive, insight-driven narrative that supports strategic decision-making.

## Data Workflow (From Raw Data to Decisions)

### 1. ETL & Data Cleaning
- Cleaned and transformed raw Excel data using **Power Query**.
- Modeled data into **Fact and Dimension tables** for optimal performance.

### 2. Data Modeling & Relationships
- Built logical relationships inside Tableau to ensure accurate calculations.

### 3. Calculations & Visualization
- Created **Calculated Fields** including **Year-over-Year (YoY) Growth**.
- Designed dashboards with a strong focus on usability and business storytelling.
